## 💡 系统架构

### 技术栈总览
```
前端：React + TypeScript + Tailwind CSS
后端：Python (Flask/FastAPI)
文档：xlsx, docx, pdf, pptx skills
测试：Playwright
工具：MCP服务器
```

### 数据流设计
```
用户输入 → Web界面 → 业务逻辑 → 文档处理 → 输出文件
```
