# Skills 学习笔记

本目录用于记录Skills学习过程中的笔记、心得和问题解决方案。

## 📂 目录结构

```
learning-notes/
├── README.md              # 本文件
├── week-1-2-basics.md     # 第1-2周：基础技能学习笔记
├── week-3-design.md       # 第3周：设计与创意学习笔记
├── week-4-5-documents.md  # 第4-5周：文档处理学习笔记
├── week-6-7-web.md        # 第6-7周：Web开发学习笔记
├── week-8-final.md        # 第8周：综合实战学习笔记
└── daily/                 # 每日学习记录（可选）
```

## 📝 笔记规范

每周的学习笔记应包含：

1. **本周学习目标** - 明确本周要掌握的skills
2. **每日学习记录** - 记录每天的学习内容和进度
3. **关键概念总结** - 重要的知识点和技术要点
4. **代码片段收藏** - 有用的代码示例
5. **问题与解决** - 遇到的问题及解决方案
6. **实战项目记录** - 项目进展和经验总结
7. **本周反思** - 学习收获和改进方向
