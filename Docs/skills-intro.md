# Anthropic Skills 技能介绍

本目录包含 **16 个 Claude 技能**，涵盖文档处理、创意设计、Web 开发、工具开发和团队协作等多个领域。

---

## 📊 **办公文档处理类**（4个）

### 1. **xlsx** - Excel 表格操作
- **作用**：全面的 Excel 电子表格创建、编辑和分析
- **核心功能**：
  - 使用 `pandas` 和 `openpyxl` 进行数据操作
  - 支持公式、格式化、数据分析和可视化
  - 包含财务模型标准
  - **重要**：带有 `recalc.py` 脚本，用于确保公式重新计算
- **典型场景**：财务报表、数据分析、自动化报告生成

### 2. **docx** - Word 文档操作
- **作用**：全面的 Word 文档创建、编辑和分析
- **核心功能**：
  - 使用 `pandoc` 进行文本提取
  - 使用 `ooxml` 进行 XML 级别访问和编辑
  - 使用 `docx-js` 创建新文档
  - **特色**："红线工作流"，支持跟踪修订
- **典型场景**：合同编辑、文档模板化、内容提取与分析

### 3. **pdf** - PDF 文档操作
- **作用**：全面的 PDF 处理工具包
- **核心功能**：
  - 使用 `pypdf` 进行基本操作（合并、拆分、元数据、旋转）
  - 使用 `pdfplumber` 进行文本和表格提取
  - 使用 `reportlab` 创建 PDF
  - 命令行工具（`pdftotext`、`qpdf`）
- **典型场景**：PDF 合并、文本提取、表单处理、加密保护

### 4. **pptx** - PowerPoint 演示文稿操作
- **作用**：演示文稿的创建、编辑和分析
- **核心功能**：
  - 从 HTML 创建演示文稿（`html2pptx`）
  - 使用模板创建演示文稿
  - OOXML 级别的编辑
  - 幻灯片重排、文本替换、缩略图生成
- **典型场景**：自动生成报告演示文稿、模板化演示、批量内容更新

## 🎨 **设计与创意类**（4个）

### 5. **algorithmic-art** - 算法艺术生成
- **作用**：使用 p5.js 创建生成式艺术
- **核心功能**：
  - 基于种子的可重现随机生成
  - 交互式参数调整
  - 算法设计哲学
  - 包含查看器模板
- **典型场景**：创意可视化、数据艺术、图案生成

### 6. **canvas-design** - 画布设计创作
- **作用**：创建 PNG/PDF 格式的视觉艺术
- **核心功能**：
  - 强调视觉表达，最少文本
  - 专家级工艺和设计哲学
  - 包含 70+ 种字体资源
- **典型场景**：海报设计、信息图、社交媒体图像

### 7. **brand-guidelines** - 品牌指南应用
- **作用**：将 Anthropic 品牌指南应用到作品中
- **核心功能**：
  - 一致的品牌色彩（橙色 #E67235、黑色 #1A1818、米白 #FAF7F2）
  - 标准排版（Poppins 用于标题、Lora 用于正文）
  - 确保品牌一致性
- **典型场景**：官方文档、营销材料、品牌相关设计

### 8. **theme-factory** - 主题工厂
- **作用**：为作品应用预设或自定义主题
- **核心功能**：
  - 10 种预设主题（Aurora、Ember、Ocean、Forest 等）
  - 色彩调色板和字体配对
  - 快速样式应用
- **典型场景**：快速设计原型、风格统一、主题化内容

## 🌐 **Web 开发与测试类**（3个）

### 9. **web-artifacts-builder** - Web 构件构建器
- **作用**：创建复杂的多组件 Claude.ai HTML 构件
- **核心功能**：
  - 使用 React + TypeScript
  - 集成 Vite/Parcel 打包
  - Tailwind CSS + shadcn/ui
  - 自动化初始化和打包脚本
- **典型场景**：交互式 Web 应用、数据可视化仪表板、复杂 UI 组件

### 10. **webapp-testing** - Web 应用测试
- **作用**：使用 Playwright 测试本地 Web 应用
- **核心功能**：
  - 自动服务器管理（`with_server.py`）
  - 无头浏览器自动化
  - "侦察-行动"模式
- **典型场景**：自动化测试、UI 验证、功能测试

### 11. **frontend-design** - 前端界面设计
- **作用**：创建生产级前端界面
- **核心功能**：
  - 高设计质量标准
  - 强调排版、色彩、动效、空间构成
  - 避免通用 AI 美学
  - 使用 React、Tailwind CSS
- **典型场景**：高品质 Web 应用 UI、着陆页、仪表板

## 🛠️ **工具与开发类**（3个）

### 12. **mcp-builder** - MCP 服务器构建
- **作用**：创建高质量的 MCP（模型上下文协议）服务器
- **核心功能**：
  - 现代 MCP 设计指南
  - TypeScript/Python SDK 支持
  - 实现、审查、评估流程
  - 包含参考文档和评估模板
- **典型场景**：扩展 Claude 功能、创建自定义工具、集成外部服务

### 13. **skill-creator** - 技能创建器
- **作用**：创建有效的 Claude 技能的指南和工具
- **核心功能**：
  - 技能设计原则（简洁性、适当的自由度）
  - 技能结构（YAML frontmatter + Markdown + 资源）
  - 六步创建流程
  - 初始化和打包脚本
- **典型场景**：创建新技能、技能模板化、技能标准化

### 14. **slack-gif-creator** - Slack GIF 创建器
- **作用**：创建针对 Slack 优化的动画 GIF
- **核心功能**：
  - `GIFBuilder` 核心工具
  - PIL 图形绘制
  - 缓动函数支持
  - 优化策略（颜色数量、帧率、尺寸）
- **典型场景**：Slack 表情符号、动画反应、团队交流 GIF

## 📝 **协作与沟通类**（2个）

### 15. **doc-coauthoring** - 文档协作
- **作用**：结构化的文档协作创建工作流
- **核心功能**：
  - 三阶段流程：上下文收集 → 精炼与结构 → 读者测试
  - 迭代优化
  - 适用于技术文档、报告、提案
- **典型场景**：协作写作、文档审查、内容改进

### 16. **internal-comms** - 内部沟通
- **作用**：内部沟通内容的指南和模板
- **核心功能**：
  - 3P 更新、公司通讯、FAQ 等模板
  - 特定文件路径的示例（`examples/` 目录）
  - 沟通最佳实践
- **典型场景**：公司公告、团队更新、内部文档
