# Skills 目录结构和作用说明

## 项目概述

这是一个 **Anthropic Skills（技能）仓库**，用于 Claude AI。Skills 是包含指令、脚本和资源的文件夹，让 Claude 能够动态加载以更好地完成特定任务。

## 📁 主要目录结构

```
skills/
├── anthropics-skills/          # 主技能仓库
│   ├── README.md              # 项目主说明文档
│   ├── THIRD_PARTY_NOTICES.md # 第三方许可声明
│   ├── spec/                  # 技能规范文档
│   │   └── agent-skills-spec.md
│   ├── template/              # 技能模板
│   │   └── SKILL.md
│   └── skills/                # 技能集合
│       ├── 文档处理技能/
│       ├── 创意与设计技能/
│       ├── 开发与技术技能/
│       └── 企业与沟通技能/
├── skills-dependencies.md     # 技能依赖说明
├── skills-intro.md           # 技能介绍
└── Docs/                     # 文档目录（本文档所在位置）
```

## 📚 详细目录说明

### 1. anthropics-skills/README.md
**作用：** 项目主说明文档

**包含内容：**
- Skills（技能）的定义和工作原理
- 如何在 Claude Code、Claude.ai 和 Claude API 中使用技能
- 如何创建自定义技能的基础指南
- 技能市场的使用方法
- 合作伙伴技能展示

### 2. anthropics-skills/spec/
**作用：** 技能规范定义

**包含文件：**
- `agent-skills-spec.md` - Agent Skills 规范文档

**规范要点：**
- 每个技能必须包含 `SKILL.md` 文件
- YAML frontmatter 必需字段：
  - `name` - 技能名称（小写，连字符分隔）
  - `description` - 技能描述和使用时机
- 可选字段：
  - `license` - 许可证
  - `allowed-tools` - 预批准的工具列表
  - `metadata` - 额外元数据

### 3. anthropics-skills/template/
**作用：** 技能创建模板

提供标准的 `SKILL.md` 模板文件，帮助开发者快速创建自定义技能。

## 📖 如何创建自定义技能

基本步骤：
1. 创建一个新文件夹
2. 在其中创建 `SKILL.md` 文件
3. 添加 YAML frontmatter（name 和 description）
4. 编写 Markdown 格式的指令和示例
5. 可选：添加脚本、资源文件等

示例结构：
```markdown
---
name: my-skill-name
description: 技能描述和使用场景
---

# 我的技能名称

[在这里添加 Claude 需要遵循的指令]

## 示例
- 示例用法 1
- 示例用法 2

## 指南
- 指南 1
- 指南 2
```
